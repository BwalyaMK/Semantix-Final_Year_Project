================================================================================
                          TROUBLESHOOTING
================================================================================

If you encounter issues:

1. Classifier Not Loading New Categories
   - Delete models/classifier.pkl
   - Delete models/vectorizer.pkl
   - Restart the application
   - System will reinitialize with new data

2. Low Confidence for New Categories
   - Add more training examples for that category
   - Use the "Add Training Example" form
   - Click "Retrain Classifier"

3. Wrong Intent Classification
   - Review training data for that category
   - Add more diverse examples
   - Check for keyword overlap between categories
   - Retrain after adding examples

4. Self-Learning Not Working
   - Check that keywords are properly defined in learning_service.py
   - Verify intent_keywords dictionary includes all 15 categories
   - Review learned data on Train page
